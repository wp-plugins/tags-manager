Tags management plugin such as Google webmaster tool, Google Analytics and Google Adsense's remarketing tag. 

== Description ==
This plugin can manage tags, such as Google Webmaster, Google Analytics and Google Adsense's remarketing.

1. You can check the tag in Google sites.
2. and then copy the tag.
3. check "Settings" in WordPress => "Tags Manager"
4. paste the tag's code in the field of each tags. 
5. show up the tags in HTML such as  (Google Analytics tags and Google Webmaster tags) or (Google Adsense remarketing tags). 


== Installation ==
1. Upload the "tags-manager" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Tags Manger" Setting menu in WordPress.
